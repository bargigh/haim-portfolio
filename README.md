# Haim Bargig Portfolio

A professional photography and cinematography portfolio with CMS backend.

## Structure
- `/frontend` - Portfolio website (GitHub Pages)
- `/sanity` - Content management system (Sanity.io)

## Features
- Photo gallery with automatic image optimization
- Video showcase with YouTube integration
- Admin backend for content management
- Responsive, clean and dramatic design
- Custom domain: haimbargig.com

## Setup
1. Deploy Sanity CMS
2. Configure frontend with Sanity connection
3. Deploy to GitHub Pages
4. Connect custom domain

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- CMS: Sanity.io
- Hosting: GitHub Pages
- Domain: haimbargig.com
